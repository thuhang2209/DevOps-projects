# 1. Private IP va Public IP khac nhau nhu the nao?
- Private IP:
    + la IP dung trong mang rieng (nha, cong ty) (router cung cap)
    + khong truy cap truc tiep tren Internet 
- Public IP
    + la IP duoc gan boi nha mang (ISP cung cap)
    + duy nhat, pham vi toan cau
# 2. DNS có vai trò gì trong việc truy cập website?
- doi domain thanh IP de browser co the hieu duoc phai gui request toi server nao 
# 3. Port 80 và Port 443 khác nhau chỗ nào?
- port 80: dung cho HTTP, khong ma hoa
- port 443: dung cho HTTPS, co ma hoa
# 4. Ngrok giải quyết vấn đề gì?
- giup public local server ra internet 
# 5. Nginx với vai trò Reverse Proxy hoạt động ra sao?
- nhan request tu user
- chuyen tiep toi server that
- tra lai res cho user
